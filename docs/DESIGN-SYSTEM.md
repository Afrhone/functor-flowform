# Flowforms design system

Seven families drive both visual form and the ODE field:

| Family | Visual grammar | Parameter field |
|---|---|---|
| Root | anchors, splits, root traces | origin wells |
| Bridge | loops, crossings, thresholds | connectivity springs |
| Form | vessels, open contours | area and proportion constraints |
| Rhythm | repeats, pulses, sweeps | sinusoidal forcing |
| Nature | branching, growth, organic balance | bifurcation and attraction |
| Momentum | direction, acceleration | curl and vector potential |
| Axiom | circles, centers, invariants | fixed centers and symmetry |

Design rule: every glyph must carry one primary family and one optional secondary behavior. This keeps the alphabet coherent while allowing playful morphics.
